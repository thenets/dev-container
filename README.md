# Dev Container

A full-featured Fedora container image designed for development.

## Use-cases

- Coder Workspace
- VS Code container (WIP)

## Features

- Most build tools. Maybe suitable for pipelines too.
- Container-in-container support:
    - [ ] Podman-in-Podman
    - [x] Podman-in-Docker
    - [ ] Docker-in-Docker
    - [ ] Docker-in-Podman
